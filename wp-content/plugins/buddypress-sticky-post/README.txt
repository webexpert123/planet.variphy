== Description ==

This plugin helps in making a buddypress post featured on the activity stream page list so that it always appears on top.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download the zip file and extract it.
2. Upload `buddypress-sticky-post` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu.
4. Enjoy
